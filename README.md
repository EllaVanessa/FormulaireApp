# FormulaireApp

Application Android permettant de saisir des informations utilisateur via un formulaire et d’afficher un récapitulatif dans un second écran.

---

## Objectif

L’objectif de cette application est de démontrer la gestion d’un formulaire ainsi que le passage de données entre deux activités en utilisant les Intents.

---

## Fonctionnalités

* Saisie des informations utilisateur :

  * Nom et prénom
  * Email
  * Téléphone
  * Adresse
  * Ville

* Validation des champs obligatoires (nom et email)

* Vérification du format de l’email

* Transmission des données entre deux écrans

* Affichage d’un récapitulatif des informations saisies

* Bouton de retour vers le formulaire

* Partage des informations via les applications disponibles (email, SMS, etc.)

---

## Architecture

L’application est composée de deux activités :

* MainActivity : écran de formulaire
* Screen2Activity : écran de récapitulatif

---

## Technologies utilisées

* Java
* Android Studio
* XML pour la conception des interfaces

---

## Fonctionnement

1. L’utilisateur saisit ses informations dans le formulaire
2. Il clique sur le bouton "Envoyer"
3. Les données sont envoyées à la seconde activité via un Intent
4. Le second écran affiche les informations sous forme de récapitulatif
5. L’utilisateur peut revenir en arrière ou partager les données

---

## Lancer le projet

1. Cloner le dépôt :

```
git clone https://github.com/EllaVanessa/FormulaireApp.git
```

2. Ouvrir le projet avec Android Studio
3. Lancer un émulateur ou connecter un appareil
4. Exécuter l’application
